## 📦 安装使用

### 安装方式
1. **VSCode/Cursor 扩展市场**：
   - 搜索 `HT Cursor Rules`
   - 点击安装

2. **手动安装 .vsix**：
   - 下载 `ht-cursor-rules-1.0.6.vsix`
   - VSCode/Cursor → Extensions → ... → Install from VSIX

### 使用步骤
1. 按 `Ctrl+Shift+P`（Mac: `Cmd+Shift+P`）
2. 输入 `HT`
3. 选择 `HT Cursor Rules: Add Cursor Rules`
4. 选择类别：`Game Development`
5. 选择规则：`Erlang 游戏服务器` 或 `Lua 游戏脚本`
6. 选择添加方式：`覆盖` 或 `合并`
7. ✅ 完成！
