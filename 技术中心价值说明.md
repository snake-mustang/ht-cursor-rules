# HT Cursor Rules - 技术中心价值说明

## 📌 工具简介

**HT Cursor Rules** 是一个 VSCode/Cursor 插件，为 Cursor AI 编程助手提供标准化的开发规则模板。支持 **28 种精选技术栈**，涵盖前端、后端、游戏开发、移动开发、测试等 **6 大类别**。

**规则来源**：基于 GitHub 最高星数的规则项目 [awesome-cursorrules](https://github.com/PatrickJS/awesome-cursorrules)（⭐ 35k+），并结合实战经验精心优化。
